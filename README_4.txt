O problema escolhido é sobre um armazém que só pode armazenar um produto.
O produto deve ser armazenado pelo produtor e retirado pelo consumidor
O produtor so pode armazenar o produto se o armazém estiver vazio
O consumidor so pode retirar se houver produto

O monitor serve para garantir essas condições, se ele não existisse poderia acontecer, por exemplo de o produtor sobreescrever um produto
que ainda não foi retirado, ou também de o consumidor tentar retirar o mesmo produto mais de uma vez, sendo assim essencial para o
funcionamento do algoritmo

Foi implementado em Java
